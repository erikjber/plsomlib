Provides extensible metrics for use by SOMs and PLSOMs.

Package Specification

Self-Organizing Maps (both SOM and PLSOM) require metrics.
Metrics in this context is a method for measuring the distance between two points in
either input space (to measure the distance between inputs and weight vectors) or output space (to measure the distance between nodes).
These measures are realized by classes in the org.plsomlib.metrics package which implement the Metric interface. An abstract implementation of this interface, MetricImpl, is provided in order to simplify subclassing.
These classes must provide functionality for XML serialization and deserialization.

There is also a default implementation that can be used to create a map with euclidean input and output spaces and a nodes arranged in a rectangular grid, EuclideanMetric.
