This package defines neighbourhood functions.

Package Specification

Neighbourhood functions determine how much the nodes close to the winning node are affected by a weight update.
These functions are realized by classes that implement the NeighbourhoodFunction interface.
There is an abstract implementation available, NeighbourhoodFunctionImpl, which also provides basic XML serialization/deserialization funcionality.
A default implementation providing a Gaussian neighbourhood function is also provided, in the class GaussianNeighbourhoodFunction.
